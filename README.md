# Patient Samurai

Simple application for managing patients. See CHANGELOG.md file to get details about development process

Features are:
- Display patient list
    - Search by patient
    - Filter by patient
- Create patient
    - Add validation
- Update patient
- Delete patient

## Stack

- Clojure
- ClojureScript
- PostgreSQL

## Editor

Sublime Text 4 with installed Clojure (Sublimed) package

## Installation

```
lein repl
```

## Tests

```
lein test
```
